Generate sample data in under 5 minutes without installing any dependencies.

### Prerequisites
- ‚úÖ Docker Desktop installed ([Download](https://www.docker.com/products/docker-desktop))
- ‚úÖ Docker Compose available (included with Docker Desktop)

### Option 1: Quick Demo Dataset (Recommended for first-time users)

```bash

# üìÅ Output: ./output/bronze/
