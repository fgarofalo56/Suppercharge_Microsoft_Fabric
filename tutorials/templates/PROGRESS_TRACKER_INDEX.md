## üîÑ Maintenance Workflow

### Adding a New Tutorial

1. **Create Tutorial README**
   - Copy structure from existing tutorial
   
2. **Add Progress Tracker**
   - Use [`PROGRESS_TRACKER_TEMPLATE.md`](./PROGRESS_TRACKER_TEMPLATE.md)
   - Update tutorial number and details
   - Set status to "TODO"

3. **Update All Existing Trackers**
   - Add new tutorial row to all tutorials (00-09)
   - Keep status as "TODO" for the new one

4. **Update Documentation**
   - Add new tutorial to this index
   - Update tutorial count if needed

### Updating Tutorial Progress

1. **Mark Current as Complete**
   - Change badge from CURRENT to COMPLETE
   - Remove background color and bold text

2. **Mark Next as Current**
   - Change badge from TODO to CURRENT
   - Add background color and bold text

3. **Update All Tutorial Files**
   - Apply changes consistently across all README files

4. **Verify Changes**
   - Preview on GitHub
   - Test links
   - Check mobile view
